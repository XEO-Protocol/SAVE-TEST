These examples model logical communication topology inspired by publicly discussed architecture patterns. They do not represent official diagrams or account for production-scale redundancy, load balancing, or infrastructure configuration.

Control planes are designed around centralised state management. SAVE will correctly flag API Server's hub dominance — this is a valid architectural tradeoff, not a flaw. The score reflects topology, not engineering quality.
